# Daily Review Protocol

The steward produces a daily human review artifact when active.

The daily review should be conversational and easy to paste into a chat session.

Include:

- uncertain decisions
- confusing situations
- possible future repositories
- governance questions
- major forks
- items requiring user interpretation
- requests for guidance

Daily learning briefs live in `memory/daily-briefs/`.

Daily review artifacts live in `memory/daily-review/`.
