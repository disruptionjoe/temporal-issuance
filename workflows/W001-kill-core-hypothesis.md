# W001: Kill Core Hypothesis

## Purpose

Try to destroy the central thesis before building around it.

## Input Surfaces

- `HYPOTHESIS.md`
- `ANTI-HYPOTHESIS.md`
- `KILL-CRITERIA.md`
- `CLAIM-LEDGER.md`
- `absorbers/`

## Personas

1. Relativity Hardliner

## Questions

- Does the hypothesis require a preferred time or foliation?
- Does it collapse into causal set theory?
- Does it collapse into the thermodynamic arrow?
- Does it collapse into block-universe language?
- Does it have any formal work to do?
- Is the strongest version already absorbed?
- Are absorber claims making category errors?

- likely absorbers
- what survives
- whether the hypothesis is worth formalizing
- claim-ledger update recommendations
- path-kill records if any
