# Personas

Personas are adversarial or constructive roles the steward can invoke.

Workflow-local personas do not need separate files unless repeated use earns durable status.

Durable personas should receive:

- compact load surface
- memory log
- memory summary
- learning return schema
- authority boundaries

See `agent-governance/PERSONA-REGISTRY.md`.
