## Result

The useful divergence is:

```text
Temporal Issuance: source-side issuance and new admissible structure.
Time as Finality: observer-shadow projection, stabilization, and record finality.
```

The report recommends preserving that split and building an explicit
source-shadow-finality interface contract instead of merging the repos or
promoting a new claim.

## Steelman Summary

The strongest constructive version is a three-layer research operating system:

1. Source layer: what can be newly issued or admitted.
2. Shadow layer: what bounded observers can access, project, compare, or lose.
3. Finality layer: what becomes stable, replayable, and hard to reverse.

This is useful only if each layer has typed boundaries and negative controls.

## Hegelian Summary

Thesis: Temporal Issuance asks where genuinely new source-side structure enters.

Antithesis: Time as Finality asks what observer-bounded projections stabilize
into records.

Synthesis: build a typed interface between source extension, projection,
capability, record finality, and loss. Do not merge the repos or import claim
authority across them.
