# Agent Runs

Every steward run should record:

- run id
- timestamp
- trigger
- workflow
- agent or subagents used
- strongest version
- strongest objection
- what collapsed
- what survived
- what was absorbed
- what was clarified
- what was promoted
- blockers
- recommended next run
- files changed
