# RUN-0003: W001 Kill Core Hypothesis

## Summary

The first manual W000 steward cycle routed to W001 because the current highest-learning move was to attack the core hypothesis before adding more infrastructure.

Result: W001 did not kill the whole hypothesis, but it did kill the naive absolute-global-frontier reading. The strongest surviving version must be local, covariant, and absorber-aware. Any future formal object must define issuance without a preferred global frontier and without smuggling in ordinary time through observer cadence.

## Strongest Version Generated

Temporal issuance, if it survives, should not be framed as a single global now-frontier moving through spacetime. The strongest version is:

> A local, observer-indexed realization-order account in which constraints become fixed relative to access, record cadence, and reconciliation rules, with any global structure reconstructed only through gluing rather than assumed as a primitive preferred frontier.

This version tries to preserve the intuition that fixedness accumulates while avoiding a direct conflict with relativity.

## Persona Findings

### Relativity Hardliner

Strongest objection:

Any literal global `dR` frontier looks like a preferred foliation. If temporal issuance requires that frontier to be physically real and globally ordered, relativity is likely a direct kill.

What survives:

A local or covariant account may survive if `dR` becomes an observer/access boundary, a domain-of-dependence boundary, or a reconstructed gluing artifact rather than a universal moving surface.

Verdict:

Kill the naive global-frontier reading. Do not kill the local reconstruction version yet.

### Block Universe Defender

Strongest objection:

If the full structure already exists, "issuance" may be a narrative imposed by embedded observers. Without a formal discriminator, temporal issuance is just a procedural story over a static object.

What survives:

The hypothesis might survive as an account of embedded access, record cadence, or constraint-realization from inside the block. But then it is no longer obviously a primitive substrate.

Verdict:

Weaken primitive-substrate language until the project can show why issuance is not just an internal perspective on a completed structure.

### Cosmologist

Strongest objection:

Cosmological expansion, horizons, and boundary conditions can create strong arrows and access structures without requiring a deeper issuance process.

What survives:

The hypothesis is not obviously killed by cosmology, but cosmology is not yet evidence for it. Issuance must be definable in non-expanding or differently expanding models.

Verdict:

Cosmology is a low-immediacy absorber compared with relativity, causal sets, block universe, thermodynamics, and information theory.

### Thermodynamics Expert

Strongest objection:

Monotone realization sounds like entropy increase, irreversibility, coarse-graining, and boundary-condition asymmetry. If `mu` tracks entropy or information, the hypothesis collapses into existing thermodynamic language.

What survives:

Temporal issuance might survive only if it distinguishes constraint fixation from entropy growth and can explain cases where record/fixedness structure and entropy accounting come apart.

Verdict:

`mu` is underdefined and should be treated as a collapse risk until W002.

### Philosopher Of Time

Strongest objection:

The hypothesis risks mixing metaphysical becoming, epistemic record access, and physical dependency structure without saying which layer owns the claim.

What survives:

The local, layer-explicit version may be research-grade if it names whether it is about ontology, observer reconstruction, mathematical representation, or physical dynamics.

Verdict:

The repo needs a layer taxonomy before claims become stronger.

### Hostile Steward

Strongest objection:

The repo could protect the idea by constantly saying "category error" whenever an absorber gets close.

What survives:

The anti-premature-closure rule is useful only if paired with real absorber mappings and path-kill records. Otherwise it becomes a defense mechanism.

Verdict:

Next run should not strengthen the hypothesis. It should map absorbers tightly enough that category-error claims become inspectable.
