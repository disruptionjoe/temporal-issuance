
The goal is to decide whether this reaches a legitimate physics bridge and, if so, what kind.

## Background

Standard asymptotic-symmetry physics establishes that BMS symmetries act at null infinity and
that soft charges and memory effects can be organized as asymptotic charges. Celestial
holography attempts to recast four-dimensional flat-space scattering data as two-dimensional
correlator-like data on the celestial sphere.

The live possibility is not the absorbed statement:

```text
ExtCat = asymptotic radiative phase space.
```

That route was already classified as absorbed by standard BMS physics.

The stronger possibility is:

```text
ExtCat = boundary category whose primitive data are celestial states/operators,
with bulk Lorentzian histories treated as a target or dual realization.
```

## Candidate Category: CelExt

Define a candidate category `CelExt`:

```text
Objects:
  boundary source states B = (Sigma, H, sector, constraints)

  where:
  - Sigma is the celestial sphere or a marked/cut version of it
  - H is a candidate celestial Hilbert/state space or operator-algebraic sector
  - sector includes soft/hard charge sector data
  - constraints encode boundary admissibility requirements

Morphisms:
  admissible boundary insertions or transition kernels
  e: B -> B'

  Examples:
  - operator insertions changing charge sector
  - celestial amplitude elements
  - boundary evolution/correlator kernels, if a boundary dynamics exists

Identity:
  neutral insertion or identity kernel on a boundary state.

Composition:
  operator composition, OPE/factorization composition, or gluing of boundary correlator kernels.

Admissibility:
  e is admissible if it respects the sector constraints, charge matching, and the boundary
  Ward identities required by the candidate celestial theory.
```

This is a candidate, not a completed construction. The main gap is that celestial holography is
not yet a fully axiomatized, universally accepted boundary CFT with a finished state space,
operator algebra, and composition law for all flat-space gravity processes.

## BMS Action

Let `G_BMS` denote the relevant BMS group or algebra, with supertranslations indexed by smooth
functions `f` on `S^2`.

Candidate action:

```text
rho: G_BMS -> Aut(CelExt)
```

where `rho(g)` acts on:

- boundary coordinates and conformal data on `S^2`;
- charge sectors;
- operator insertions;
- Ward-identity constraints.

For this to be a genuine action on `CelExt`, the following must hold:

```text
rho(g)(id_B) = id_{rho(g)B}
rho(g)(e2 o e1) = rho(g)(e2) o rho(g)(e1)
e admissible iff rho(g)(e) admissible
```

This is plausible at the kinematic symmetry level because BMS is the organizing symmetry of the
celestial/asymptotic structure. It is not yet proven as a complete functorial action on a finished
`CelExt`, because the finished category is not currently supplied by Temporal Issuance.

## Five-Reviewer Synthesis

### 1. Category Theorist

The route becomes coherent only after replacing generic `ExtCat` with a structured boundary
category. A functorial BMS action is plausible if morphisms are boundary operators or kernels and
composition is OPE/gluing/factorization. It is not a theorem until the category and admissibility
predicate are specified.

### 2. Lorentzian / Celestial Geometer

The construction is not a Lorentzian metric reconstruction. It uses the celestial sphere and
asymptotic symmetry structure supplied by flat-space physics. The route naturally reaches
conformal boundary data before it reaches metric data.

### 3. Relativity Physicist

This is the strongest physics-facing bridge because BMS charges and memory are real observables
in asymptotically flat gravity. But the physics comes from asymptotic symmetry and celestial
holography, not from the current Temporal Issuance axioms.

### 4. GU Specialist / Skeptic

This does not yet help GU specifically. It points toward boundary/asymptotic symmetry geometry,
not GU observerse geometry. GU should remain outside the construction unless a later run shows
that the boundary category naturally embeds in a GU-style observer/field geometry.

### 5. Philosophy Of Science Reviewer

The earned claim is a conditional bridge, not a derivation. The main danger is equivocation over
"source": boundary source data in celestial holography is not the same thing as Temporal Issuance
source-state data unless an explicit identification is added and tested.
