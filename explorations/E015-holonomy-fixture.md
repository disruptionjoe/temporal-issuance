# E015: Holonomy Fixture for Source-Generated Connection

## Question

Can a minimal typed constraint system `(C_min, <=_S^min, Ext_S^min)` with a closed extension loop
determine a nontrivial `G`-valued holonomy from `Ext_S` alone, without importing a target gauge
theory, Berry phase, BMS connection, GR connection, or other physical connection?

## Verdict

```yaml
bare_ext_s_derives_nontrivial_holonomy: false
transport_enriched_ext_s_has_nontrivial_holonomy: true
physics_derived: false
claim_effect: TI-C018_weakened
path_kill: bare_Ext_S_to_nontrivial_holonomy_without_transport_data
next_fixture: cech_sheaf_fixture
```

The fixture gives a clean formal result and a clean negative result.

Formal positive result: if `ExtCat` is enriched with a transport functor
`A: ExtCat -> B G`, then closed extension loops can carry nontrivial holonomy invisible to the
induced preorder.

Negative result: bare `Ext_S` plus a closed loop does not determine a group element. The loop
word is not a holonomy until a representation, transport rule, or connection is added.

## Load-Bearing Distinction

There are two different objects that must not be conflated.

### Bare Extension Category

`ExtCat_bare` contains:

- objects: typed source states;
- morphisms: admissible extensions;
- composition: sequencing of extensions;
- thin reflection: the induced preorder `<=_S`.

This supplies loop shape, parallel morphisms, and non-order category structure. It does not by
itself supply a value in a group `G`.

### Transport-Enriched Extension Category

`ExtCat_A` contains all of `ExtCat_bare` plus:

```text
A: ExtCat_bare -> B G
```

where `B G` is the one-object category whose morphisms are elements of `G`. In concrete terms,
every elementary extension `e` receives a transport value `A(e) in G`, and composition obeys:

```text
A(e2 o e1) = A(e2) A(e1)
```

Only `ExtCat_A` has holonomy. The central fixture question is whether `A` is derived from
typed source admissibility or merely attached to it.

## Minimal One-Loop Fixture

Let `ExtCat_0` be generated by:

```text
object: S
morphism: ell: S -> S
relations: none beyond category identities
```

The induced preorder has one object and no distinction:

```text
Preord(ExtCat_0) = {*}
```

Now choose a group `G`.

A holonomy assignment is exactly a functor:

```text
A: ExtCat_0 -> B G
```

Such functors are in bijection with choices of one group element:

```text
A(ell) = g in G
```

Then:

```text
Hol_A(ell^n) = g^n
```

If `g != 1_G`, the holonomy is nontrivial and not recoverable from the induced preorder.
But `ExtCat_0` does not choose `g`. The same bare extension category supports every assignment
`g in G`.

Therefore:

```text
closed_loop + bare_Ext_S != nontrivial_holonomy
closed_loop + chosen_transport_functor = formal_holonomy
```

This is the exact place where the claimed source-generated connection must appear.

## Three-Patch Source Fixture

The strongest attempted source-side version uses a minimal patch system rather than a pure
one-loop category.

Define `C_min` with:

```yaml
patches: [U0, U1, U2]
local_frames:
  - f0 over U0
  - f1 over U1
  - f2 over U2
elementary_compatibilities:
  - t01: f0 -> f1
  - t12: f1 -> f2
  - t20: f2 -> f0
loop: t20 o t12 o t01
```

If each `tij` is assigned a value in `G`, then:

```text
Hol(loop) = A(t20) A(t12) A(t01)
```

This can be nontrivial. For `G = U(1)`, for example:

```text
A(t01) = exp(i alpha)
A(t12) = exp(i beta)
A(t20) = exp(i gamma)
Hol(loop) = exp(i (alpha + beta + gamma))
```

But the fixture has only three possible outcomes.

1. `C_min` does not include the transition values. Then no holonomy is determined.
2. `C_min` includes transition values and allows nonidentity product. Then the holonomy is
   formal but the transport data has been stipulated.
3. `C_min` includes a consistency rule forcing the product to be identity. Then the holonomy is
   trivial.

No fourth option appeared in the minimal fixture.

## Strongest Surviving Version

The strongest surviving version of TI-C018 is:

```text
If typed source admissibility includes or independently derives a transport functor
A: ExtCat -> B G, then extension loops can carry a non-order morphism invariant
Hol_A(gamma) in G.
```

This is a correct formal result. It strengthens TI-C008 as formal residue: extension categories
can carry invariants not recoverable from the induced preorder.

It does not establish that current Temporal Issuance primitives derive the connection.

## Destruction Test

The fixture attacks the strongest version from first principles.

### Test 1: Bare Loop

Result: fail.

A closed loop in `ExtCat` determines a loop word, not a `G`-valued holonomy. The group element
appears only after choosing `A`.

### Test 2: Source-Typed Transition Labels

Result: conditional formal success, but not derivation.

If transition labels are made part of `C_min`, holonomy is determined by the extension rule.
However, the label system is exactly the transport/connection data whose independent derivation
was under test. This does not import target physics, but it does import extra mathematical
structure.

### Test 3: Boundary Observable

Result: no physics claim.

A Wilson-loop-style boundary observable distinguishes the enriched cases only after a connection
or transport functor exists. If the observable is physical, the physical meaning comes from the
theory that interprets the connection.

## Absorber Assessment

The formal holonomy result is absorbed by standard mathematics unless a source-side admissibility
predicate forces the transport.

Nearest absorbers:

- path groupoids and functors into `B G`;
- representations of fundamental groupoids;
- principal `G`-bundles with connection;
- Cech cocycles and transition functions;
- gauge theory if the connection is physically interpreted.

Temporal Issuance residue remains only if the transport values are forced by typed admissible
extension rules rather than selected as an independent representation.

## Claim Effect

TI-C018 should be weakened, not promoted.

It is not killed in its formal conditional form. It is killed as a claim that bare `Ext_S`
derives nontrivial holonomy. The exact survivor is transport-enriched formal residue.
