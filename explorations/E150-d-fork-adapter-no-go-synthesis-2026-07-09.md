## Candidate No-Go Schema

Bounded schema, not yet a theorem:

```text
For a physical candidate trace routed through Adapter_P:

1. If the trace factors through a fixed completed source plus an access/readout/finality
   schedule that preserves tau_n, the trace is fixed-source disclosure.
2. If the trace defeats finite-prefix precontainment but is absorbed by a completed realized
   family or singleton enumerator after the fact, it earns at most the class-relative
   OnlineIssuance^LC residue.
3. If the trace avoids both only by importing a global table, law change, boundary category,
   or modeler schema, it is imported structure.

Therefore a physical source-issuance winner must supply an internally specified rule that
blocks after-naming without smuggling in a hidden completed source.
```

This is the useful next research shape: not "find more novelty language," but state and test
the exact class over which the no-go schema can be made precise.
