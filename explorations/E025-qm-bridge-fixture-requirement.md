# E025: QM Bridge — Fixture Requirement

## Idea Tested

```text
Quantum superposition = the formal expression of the gap between:
  (a) what the observer can compute from current schema S_n (their predicted future states)
  (b) what issuance actually produces (D4 type-novel events outside S_n)
Wavefunction collapse = the moment issuance commits to a specific D4 extension, making it
accessible to the observer's schema for the first time.
Core claim: Superposition is not "multiple states simultaneously" but the observer's
schema-predictive underdetermination of a VDF-style computationally unavailable future event.
```

## What Is Killed

1. **Direct claim: superposition IS schema-predictive underdetermination.**
   Kills: D4 type-novelty is underdetermination-of-type; QM superposition is
   underdetermination-of-eigenvalue within a fixed type. These are different categorical
   levels. No bridging derivation was produced or outlined. [Category Error Auditor, verdict: killed]

2. **VDF-strong claim: quantum state transition = VDF-style computational unavailability.**
   Kills: VDF outputs are deterministic given input (T sequential steps to a fixed answer).
   Quantum outcomes are not determined by anything under standard no-hidden-variable
   interpretations. Mapping VDF onto QM conflates "I cannot compute it yet" with "there is
   no fact to compute." [VDF complexity theorist]

3. **Any QM bridge without first establishing TI-C020.**
   Kills: The physical universe as OnlineSchemaSys is speculative and unearned. The direct
   BDO/ICO path from TI primitives to physics is closed. Any QM bridge presupposes TI-C020.
   Building the bridge before B1/HoTT constructive derivation is premature by the repo's
   critical-path ordering. [Hostile Steward, Category Error Auditor]

4. **Wavefunction collapse = D4 schema-extension event (as stated).**
   Kills: Cannot recover Born-rule probability weights, complex Hilbert space structure, or
   interference terms from OnlineSchemaSys primitives. Collapse within a fixed Hilbert space
   is stochastic update (D4 Case 2, already excluded). [Decoherence physicist, MWI expert]

5. **The epistemic-only version.**
   Kills: If hidden variables are admitted (Bohmian mechanics), observer underdetermination is
   purely epistemic — which every hidden-variable theory already asserts without needing D4.
   [Bohmian mechanics expert, verdict: absorbed]

## What Survives (Narrow Residue)

### Surviving Claim 1: The H_fixed vs. H_growing Distinction
The formal distinction between (a) outcome-surprise within a fixed, pre-given Hilbert space
and (b) schema-expansion events where the outcome type was not expressible in the prior schema
S_n is **real** and **not captured by any existing QM interpretation**. All existing
interpretations — Copenhagen, QBism, RQM, MWI, decoherence, Bohmian, consistent histories,
topos QM — work inside a fixed observable algebra or Hilbert space. D4 type-novelty would
apply only to physical processes where the Hilbert space itself is subject to type-novel
extension. This distinction is not currently exploited by any QM formalism.

**Caveat:** This surviving claim requires TI-C020 to be at minimum plausible. Without
TI-C020, it is a structural observation about what TI could contribute if TI-C020 were
established.

### Surviving Claim 2: Static vs. Growing Context Category
Topos QM (Isham-Döring) uses a **static** context category (lattice of commutative
subalgebras of a fixed B(H)). OnlineSchemaSys with NAA posits a **growing** context
category where new contexts become available through issuance. This is a structural
difference that static topos QM does not cover. If TI's growing context-category can be
formally specified with NAA constraints, it provides a publishable structural observation
connecting OnlineSchemaSys to sheaf-theoretic QM — independent of whether the physical
bridge goes through.

### Surviving Claim 3: Assembly Theory Operationalization Pointer
Assembly Theory provides an independent operationalization candidate for D4 that is absent
from any QM framework. D4 events correspond (potentially) to objects with no assembly path
through the current object space — undefined assembly index relative to the prior schema.
If this equivalence can be formalized, it gives D4 an empirically testable criterion that
QM frameworks do not possess. [See `explorations/E023-crypto-econ-intake-synthesis.md`,
B2 (Assembly Theory D4 operationalization)]

### Surviving Claim 4: VDF in Weakened Form
The forward production of a D4 event from S_n is not reversible under NAA — operations at
prefix n cannot pre-compute the extension. This is structurally analogous to VDF
sequentiality (T steps cannot be shortened), even without cryptographic identity. The VDF
strong claim (quantum randomness = VDF computation) is killed; the weaker form (NAA enforces
forward-computational irreversibility structurally similar to VDF) survives at the level of
the formal description of OnlineSchemaSys.

## What the Fixture Requires

The synthesis verdict is `needs_fixture_before_verdict`. The fixture:

**Required:** Exhibit one candidate physical measurement event where:
1. The pre-measurement observable algebra A_n and the post-measurement observable algebra
   A_{n+1} are demonstrably **non-isomorphic** in a way that tracks D4 type-novelty (a new
   type enters the algebra, not merely a new eigenvalue is selected within it).
2. The D4 criterion does work that Born-rule calculation on a fixed H cannot do on its own.
3. Assembly Theory's assembly-index-undefined criterion is testable for that candidate event.

**Candidate physical settings to test:**
- Quantum gravity / topology-changing transitions (spacetime adds degrees of freedom)
- Early-universe electroweak phase transition (new particle species appear; particle physics
  standard model type-space expands in practice — whether this is a Hilbert-space extension
  or a ground-state shift within a fixed H needs examination)
- Quantum error correction that introduces new logical qubit types not definable in prior code
  space (code space expansion)
- Extended Wigner's friend scenarios (Frauchiger-Renner): if observers are assigned
  incompatible facts, reconciliation may require extending the joint Hilbert space

**Kill condition for this fixture:** If every candidate physical measurement can be embedded
in a fixed H from the start — that is, if no physical process ever forces Hilbert space
extension in the D4 sense — then the idea is killed as absorbed by epistemic QM
interpretations and should be archived.

**Promotion condition:** If a concrete fixture is found where D4 does work that Born-rule
calculation on a fixed H cannot replicate, promote to a new speculative claim adjacent to
TI-C020.

## Projection Risks (from Synthesis)

1. **Anthropomorphizing the wavefunction:** superposition is an objective interference
   phenomenon (measurable via fringes, Bell inequalities) — not an observer's cognitive
   limitation. Conflating "observer cannot predict" with "state is underdetermined" is a
   category error.

2. **Level-of-description confusion:** D4 is type-theoretic (schema extension); QM
   superposition is algebraic (vectors in Hilbert space). Running them together without a
   level-crossing derivation is a category error.

3. **VDF over-extension:** VDF's T sequential steps = complexity-theoretic claim; NAA's
   no-future-oracle = information-boundary claim. Structurally similar, not identical.

4. **Premature TI-C020 assumption:** every step of the QM bridge implicitly treats TI-C020
   as established.

5. **AB contextuality confusion:** AB obstruction = no global section in fixed context
   category; D4 = context category itself grows. Different theorems at different structural
   levels.
