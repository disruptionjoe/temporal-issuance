# E124: Relativistic Block-Universe Absorber Gauntlet

## Purpose

Execute E122 Gauntlet A:

```text
Does ongoing issuance survive the relativity-of-simultaneity argument, or is
it absorbed into fixed block-universe structure described procedurally, or
into an existing covariant-becoming program such as causal-set growth?
```

This is an absorber gauntlet, not a promotion pass. It tests whether the live
Temporal Issuance language still has source-side work after removing every
global-present, preferred-foliation, and objective-passage reading.

## Strongest Relativistic Objection

The strongest form of the objection is:

```text
If "ongoing issuance" means that new possibilities enter reality along a
single global present, then the hypothesis requires a distinguished
simultaneity structure. Special relativity forbids that structure. General
relativistic covariance makes the same demand even more expensive.
```

That objection succeeds against the weak reading. A global frontier, universal
tick, or privileged slice is not a viable source-side primitive for this repo.
This agrees with E095: coherent issuance cannot mean synchronized by a shared
clock.

## Effect Typing

| Relativistic / block-universe element | Effect typing | Reason |
| --- | --- | --- |
| Fixed four-dimensional spacetime completion | external-completion / fixed-source absorber, not `Issue[S]` | If every event and relation is already part of the completed structure, describing slices or local updates procedurally does not form new source-side admissibility. |

## Exit A: Observer / Region-Indexed Issuance

The most charitable surviving version is not global becoming. It is:

```text
An observer- or region-indexed source extension changes the task menu available
to that region, without requiring a privileged foliation.
```

This survives only as a burden, not as evidence. E119 already states the right
form of that burden:

```text
Find an Ext_S rule that changes a region-indexed observer task menu in a way
TaF's capability-typed readout cannot express; otherwise archive the surplus
as readout/projection/finality vocabulary.
```

The relativistic gauntlet sharpens this burden: the task-menu change must be
locally or regionally stated and must not depend on a hidden global slice.

## Exit B: Covariant Local Becoming / Causal-Set Growth

The second exit is a covariant growth model: events are added in a way that is
independent of any chosen linear extension. This is a legitimate comparison
class. It also absorbs much of the naive issuance language.

To beat this absorber, `Issue[S]` would need at least one of the following:

1. A formed admissibility predicate not already supplied by the CSG dynamics.
2. A source-side construction-space extension not reducible to stochastic
   element addition under a fixed law.
3. A region-indexed capability difference that remains after comparing against
   fixed block completion, fixed source plus changing access, and TaF
   capability readout.

This run does not find such a witness. Therefore CSG-style covariant becoming
remains an undefeated absorber for any "issuance is just covariant event
growth" reading.

## E121 Boundary

E121's `no_fixed_point_of_absorption` theorem is not a relativity result, not a
spacetime theorem, and not a refutation of block-universe structure.

The honest contact is only a quantifier-order analogy:

```text
For a total Nat-indexed family, a diagonal can escape that family; a fixed
extension can absorb that diagonal; the extension has its own escaping
diagonal.
```

That does not imply:

```text
No fixed spacetime block can contain all events.
```

The strict statement for this gauntlet is weaker and cleaner:

```text
Any attempted source-issuance reading must show a locally meaningful
admissibility or capability surplus that is not already captured by fixed
spacetime completion, CSG-style covariant growth, or observer readout/finality.
```

## What Collapsed

- Any global-present or preferred-foliation reading of issuance.
- Any objective-passage reading of `Issue[S]`.
- Any claim that local record availability by itself implies source issuance.
- Any causal-set-like growth story that adds no surplus beyond covariant event
  addition under a fixed law.
- Any citation of E121 as if it were a relativity or spacetime theorem.

## What Survives

`OnlineIssuance^LC` survives unchanged as a local constructive formal residue.
It is not a physical spacetime claim.

The only physics-facing residue after this gauntlet is a burden:

```text
State a locally covariant or region-indexed Ext_S rule that changes a
capability/task menu in a way not expressible by fixed block completion,
CSG-style growth, fixed-source aperture disclosure, or TaF capability readout.
```

That is not claim movement. It is a narrower next test.

## What Would Reverse This Verdict

Reopen the relativistic branch only if a future fixture supplies all of the
following:

- no preferred foliation or global present;
- a concrete observer/region support condition;
- a task-menu or capability difference;
- a source-side admissibility or construction-space change;
- comparison against fixed spacetime completion, CSG growth, fixed-source
  aperture disclosure, and TaF capability readout;
- a preserved record trace that lets the comparison be audited.

Absent that package, relativity and block-universe completion absorb the
physical reading.
