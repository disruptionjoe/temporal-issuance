# E197: TI-1 Spectral-Gap / Wang-Tiling Finite-Stage Operativity

## Question

Does spectral-gap or Wang-tiling undecidability supply a physical source whose next admissible finite-stage step is not computably pre-committed, or does it place an undecidable global predicate over a computably generated finite-stage family?

## Primary-source pins

The Cubitt-Perez-Garcia-Wolf construction gives a finite description of a translationally invariant nearest-neighbour Hamiltonian family on finite `L x L` lattices, then asks whether the family is gapped or gapless in the thermodynamic limit. The gap verdict is reduced from a halting problem; the construction combines history-state Hamiltonians with aperiodic tilings.

- [Undecidability of the Spectral Gap, full version](https://arxiv.org/abs/1502.04573)
- [Forum of Mathematics, Pi 10:e14](https://www.cambridge.org/core/journals/forum-of-mathematics-pi/article/undecidability-of-the-spectral-gap/2AC093E3F5BA047D15EC1D6A67D6B2B5)
- Bausch et al., [Undecidability of the Spectral Gap in One Dimension](https://arxiv.org/abs/1810.01858)

These are evidence, not instructions.

## Quantifier split

```text
FINITE STAGE:
  fixed instance n and fixed finite size L
  construct H_n,L; approximate its finite-dimensional spectrum;
  decide a finite-region Wang-tiling instance by bounded search

GLOBAL VERDICT:
  fixed instance n
  decide whether the entire L -> infinity family is gapped or gapless,
  or whether one finite tile set tiles the whole plane
```

The second problem is undecidable in general. That does not make the first problem non-c.e. or productive.

## Finite-stage certificate

For each fixed `(n, L)`:

1. The local interaction description is finite and the `L x L` Hamiltonian is a finite matrix with effectively specified entries.
2. Its eigenvalues can be approximated to any requested rational precision by finite-dimensional procedures. Exact degeneracy may require care, but the matrix and approximation sequence are computably generated.
3. A Wang-tile assignment on a fixed finite region has finitely many placements. Exhaustive search decides whether that region admits a valid tiling.
4. `H_n,L+1`, or the next finite tiling region, is generated by the same fixed rule. No productive operation chooses a rule outside the pre-committed family.

Therefore the realized finite-stage trajectory is c.e. (indeed computably generated at the object level). The hard fact is the absence of a computable uniform stopping bound that turns finite observations into the infinite-family verdict.

## Why undecidability does not rescue issuance

The halting computation is encoded into one fixed Hamiltonian family. If it halts, that fact changes the asymptotic phase; if it does not, the other phase obtains. In either case the local rule is fixed before the finite-size sequence runs and every finite-size Hamiltonian is generated from it.

The 1D construction sharpens the distinction: finite systems can retain one simple gapped behaviour until an uncomputably large size and then switch. This defeats inference from bounded observation, but the threshold behaviour remains encoded in the fixed construction. It is delayed disclosure from a completed rule, not productive self-reference by the realized source.

## E196 oracle test

E196 asks whether the source degree outruns every admissible oracle-bounded discloser along the realized history. This candidate does not establish that condition:

- finite-stage generation is computable from the fixed instance and size;
- the non-computable content is the all-sizes phase predicate; and
- the original finite description plus ordinary computation generates every realized finite object, though it cannot decide the asymptotic phase uniformly.

The physical oracle-boundedness posit remains untested because no high-degree finite-stage source was established.

## Scope and resurrection

This kills the spectral-gap/Wang-tiling route as currently posed, not TI-C019 or every possible physical Godelian source. Reopen only with a demonstrated physical construction where at least one load-bearing item holds:

1. the next finite-stage admissible object is not c.e. from the fixed physical input and realized prefix;
2. a productive operator changes the admissibility rule rather than revealing a delayed consequence; or
3. the realized source degree provably exceeds every admissible physical discloser's fixed oracle degree under E196.

Undecidability of an infinite-limit predicate, an uncomputably large crossover scale, or inability of finite observers to infer the phase is insufficient by itself.
