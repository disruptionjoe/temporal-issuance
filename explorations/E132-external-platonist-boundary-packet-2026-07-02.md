## External Platonist Absorber

Define the external Platonist absorber for current repo purposes:

```text
ExternalCompletion(Trace) :=
  a stance or model in which the whole source trace, including future
  admissible candidates and witnesses, exists as a completed structure outside
  the local constructive source context.
```

This absorber is not defeated by `OnlineIssuance^LC` because the Lean object
language intentionally proves an internal exclusion:

```text

```

It does not prove:

```text

```

argument outside the current bounded Lean fixture.
