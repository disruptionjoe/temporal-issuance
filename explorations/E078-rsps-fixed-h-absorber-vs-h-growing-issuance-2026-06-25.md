# E078: RSPS Fixed-H Absorber vs H-Growing Issuance Fixture

## Purpose

Execute Goal 4 of the five-goal disproof ladder:

> Test whether fixed-H quantum mechanics plus decoherence / Quantum Darwinism can reproduce
> every observer-accessible RSPS trace, or whether any trace requires H-growing or A-growing
> Temporal Issuance.

## Accessible RSPS Trace Vocabulary

The RSPS line produces observer-accessible traces of this form:

```text
Trace_RSPS =
  B*                    selected pointer basis / history family
  F(B*)                 record-fidelity score
  R_delta(B*)           redundancy / accessible fragment profile
  D(B*)                 decoherence or agreement status
  p_i                   weights, supplied by trace rule or separate module
  G_record              record graph / finality graph
  O_access              observer-access map over fragments / records
```

After RUN-0070 through RUN-0072, this trace has a strict boundary:

```text
F selects B*
F does not derive p_i
p_i = Tr(rho Pi_i) or another explicit probability module
```

## Fixed-H Null Model

The fixed-H absorber supplies:

```text
H_total = H_S tensor H_E
A_total = B(H_total) or a fixed C*-observable algebra
rho_0   = initial state on H_total
U_t or Phi_t = fixed unitary / CPTP / Lindblad dynamics
M_O,n   = observer-access instrument or record map at stage n
Pi_i    = pointer projectors selected by dynamics/environment
p_i     = Tr(rho_S Pi_i)
```

Apparent growth is represented as:

```text
A_access,n = M_O,n(A_total)
```

or as an increasing family of accessible subalgebras / records inside a fixed `A_total`.

This can model:

1. Controlled-copy / GHZ record formation.
2. Imperfect copying.
3. Partial environment access.
4. Non-orthogonal environment records.
5. Noise.
6. Varying number of fragments.
7. Pointer-basis selection.
8. Redundancy loss or degeneracy when no accessible record remains.
9. Born weights from the trace rule.
10. Observer finality as stable record accessibility.

## H-Growing / A-Growing Alternative

The Temporal Issuance alternative would need at least one witness:

```text
W1: non-isomorphic Hilbert spaces H_n -> H_{n+1} not representable inside fixed H_total
W2: non-isomorphic observable algebras A_n -> A_{n+1} not representable as accessible subalgebras
W3: new admissibility predicate not definable from the fixed dynamics/instrument
W4: construction-space growth not representable as fixed source plus access maps
W5: perturbation non-factorization through fixed channels
W6: observer record preservation under the above witness
```

These are the same kind of gates used in E057/RUN-0052.

## Absorber Test

| RSPS trace component | Fixed-H representation | H-growing residue? |
|---|---|---|
| selected pointer basis `B*` | pointer projectors from system-environment dynamics / predictability sieve | no |
| fidelity score `F(B*)` | scalar functional over candidate bases inside fixed `H_total` | no |
| redundancy profile `R_delta` | mutual information / fragment accessibility inside fixed environment | no |
| decoherence/agreement `D(B*)` | suppression of off-diagonal terms under fixed CPTP/Lindblad dynamics | no |
| Born weights `p_i` | trace rule `Tr(rho Pi_i)` | no |
| record graph `G_record` | observer-accessible records generated by fixed dynamics and instruments | no |
| changing access over time | increasing accessible subalgebras / fragments `A_access,n` | no |
| no-record degeneracy | `M_O,n` has no distinguishing record or channel is fully noisy | no |
| nonorthogonal records | POVM / Helstrom distinguishability within fixed `H_total` | no |
| noise / imperfect copying | fixed noisy channel / Lindblad process | no |

## Interpretation

The RSPS record/finality layer is absorbed as standard fixed-H quantum reconstruction
vocabulary:

```text
basis selection = decoherence / einselection / predictability sieve / record redundancy
weights = trace rule or separate probability module
observer finality = stable access to records/fragments
```

There is no surviving H-growing or A-growing residue in the RSPS line as currently formulated.

## What Would Resurrect The Route

The RSPS route could reopen only if a future candidate supplies at least one of:

1. A record trace whose observable algebra is non-isomorphic to any accessible subalgebra of a
   fixed `A_total`.
2. A measurement/finality event whose admissibility predicate cannot be represented by fixed
   dynamics plus observer instruments.
3. A construction-space event not precontained in fixed `H_total`, fixed `A_total`, or fixed
   `Mu_infty`.
4. A perturbation that provably does not factor through any fixed CPTP/open-system channel.
5. A preserved observer record showing the above difference without changing the trace
   vocabulary after the fact.
