# Absorbers

An absorber is an existing framework that may already explain the Temporal Issuance intuition.

Absorption is not automatically a stop signal. Each absorber file should ask:

- what it explains
- what it does not explain
- whether the comparison category is correct
- whether anything remains
- what would resurrect the path
