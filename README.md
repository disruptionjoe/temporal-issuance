# Temporal Issuance

Temporal Issuance is a public research-governance repository for testing a speculative hypothesis:

> The deepest global substrate may not be matter, energy, information, spacetime, or records, but temporal issuance: the monotone realization of constraint from earlier to later states.

A stronger mathematical guess is:

> Reality may be modeled as a growing measured realization order with observer-indexed record cadences.

This repo is not here to protect that idea. It exists to determine, as efficiently as possible, whether the idea should be killed, absorbed by existing theory, narrowed into a formal object, converted into testable claims, promoted after hostile review, or archived as useful but non-research-grade intuition.

## What This Is

- A research program organized around adversarial hypothesis stewardship.
- A memory-bearing steward system, not just a folder of documents.
- A place to generate the strongest surviving version of the hypothesis, then attack it from first principles.
- A governance experiment whose methods should become reusable in future research repos.

## What This Is Not

- Not a finished physical theory.
- Not a replacement for relativity, quantum theory, thermodynamics, cosmology, causal set theory, information theory, or philosophy of time.
- Not an attempt to prove the founding intuition correct.
- Not a physics-crank manifesto.
- Not merged with `time-as-finality`.

## Steward Architecture

The Repo Steward is an anchored agent with durable memory. It persists through repository state rather than model continuity.

The steward can:

- select work
- create dynamic workflows
- revise or retire workflows
- create dynamic personas
- attach memory packs to workflows or personas
- launch parallel subagents when surfaces are separable
- update claim status
- maintain roadmap and memory
- record killed paths and exportable ideas

The hourly trigger should call the steward first. The steward then chooses or creates the workflows and personas most likely to increase learning velocity.
